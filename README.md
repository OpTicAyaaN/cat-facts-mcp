```sh-session

$ mcp (--version)
```

## `mcp`

Update installed plugins.

```
USAGE
  $ mcp  [-h] [-v]

FLAGS
  -h, --help     Show CLI help.
  -v, --verbose

DESCRIPTION
  Update installed plugins.
```
